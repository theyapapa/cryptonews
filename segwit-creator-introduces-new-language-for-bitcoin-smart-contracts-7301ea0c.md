# SegWit 크리에이터는 비트 코인 스마트 계약에 대한 새로운 언어를 소개합니다 ...

###### 2019-08-21 08:08

비트 코인 코어 개발자 피터 Wuille는 새로운 비트 코인을 도입했다 (BTC) 스마트 계약 프로그래밍 언어 더빙 미니 스크립트.

Wuille, 주요 비트 코인 네트워크 업그레이드 SegWit의 창조자, 8 월 19 일에 비트 코인 개발자를위한 메일링 리스트 메시지에 미니 스크립트 프로젝트 웹 사이트를 발표했다.

## 미니 비트코인 스크립트

미니스크립트는 구조화된 방식으로 비트코인 스크립트의 하위 집합을 작성하는 언어로, Wuille가 간결하게 설명하는 것처럼 분석, 구성, 일반 서명 등을 가능하게 합니다.

한편, 비트코인 스크립트는 사용자가 특정 거래를 완료하기 위한 특정 조건을 설정할 수 있도록 하여 비트코인 네트워크에서 스마트 계약을 가능하게 하는 기본 프로그래밍 언어입니다.

Wuille에 따르면, 미니 스크립트 언어의 채택은 스마트 계약에 사용되는 소프트웨어의 조각 사이의 일부 장벽을 줄일 수 있습니다, 또한 간단한 스크립트 언어보다 효과적인 방법으로 작동하는 multisig 생태계를 제공.

## 스크립트 분석이 더 쉬워졌습니다.

미니 스크립트는 기본적으로 비트 코인 스크립트의 재구성이지만, 그것은 동일한 스크립트 언어로 남아 있지만, 대신 지갑 및 기타 소프트웨어가 보다 효과적으로 스크립트를 구성하고 분석 할 수 있습니다, Wuille는 레딧에 설명했다.

게시물에서 Wuille는 미니 스크립트 언어의 하나와 비교하여 기본 비트 코인 스크립트에서 동일한 표기법을 어떻게 보이는지 예를 제공했습니다.
