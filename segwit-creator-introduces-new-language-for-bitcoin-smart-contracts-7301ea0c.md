# SegWit Creator представляет новый язык для смарт-контрактов Bitcoin ...

###### 2019-08-21 08:08

Разработчик Bitcoin Core Питер Вуй (Peter Wuille) представил новый язык смарт-контрактов Bitcoin (BTC), получивший название Miniscript.

Wuille, создатель крупного обновления сети Bitcoin SegWit, объявил о веб-сайте проекта Miniscript в рассылке сообщений для разработчиков Bitcoin 19 августа.

Скрипт мини-биткойна

(Альт-Н)

Miniscript — это язык для написания подмножеств скриптов Bitcoin структурированным образом, который позволяет анализировать, композицию, общее подписание и другие, как вкратце описывает Wuille.

Между тем, Скрипт Bitcoin является основным языком программирования, который делает возможными смарт-контракты в сети Bitcoin, позволяя пользователям устанавливать конкретные условия для определенной транзакции.

По словам Вуйя, принятие языка Miniscript может уменьшить некоторые барьеры между частями программного обеспечения, используемого в смарт-контрактах, а также предоставление мультисиг экосистемы, которая функционирует более эффективным способом, что простой язык скриптов.

Анализ сценариев упростил

(Альт-Н)

В то время как Miniscript в основном является переформулированием сценария Bitcoin, он остается тем же языком скриптов, но вместо этого позволяет кошелькам и другому программному обеспечению более эффективно создавать и анализировать скрипты, объяснил Вуй на Reddit.

В посте, Wuille представил пример того, как та же нотация будет выглядеть в базовом сценарии Bitcoin по сравнению с одним на языке Miniscript.
