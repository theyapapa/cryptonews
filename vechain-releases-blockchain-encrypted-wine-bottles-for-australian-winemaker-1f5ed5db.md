# VeChain выпускает блокчейн-шифровированные бутылки вина для австралийского винодела ...

###### 2019-08-06 19:08

Блокчейн-платформа veChain в партнерстве с виноделом Autralian Penfolds выпустила на продажу корпус для блокчейн-шифровок винных бутылок в рамках своей инициативы По отслеживанию вина (WTP).

Блокчейн для вина 

(Альт-Н)

Ряд компаний начинают выпускать системы проверки блокчейна на вино.

Как ранее сообщал Cointelegraph, аудиторская фирма «Большой четверки» Ernst и Young объявила, что блокчейн-решение Ethereum будет использоваться для проверки подлинности импортируемых европейских вин в Азии.

Это решение, как сообщается, будет реализовано на платформе электронной коммерции Татуировка, для использования Blockchain Wine Pte. Ооо.

Сообщается, что Overstock также намеревался бороться с винным мошенничеством, но в данном случае с помощью разработки цифровой торговой платформы для винных фьючерсов.

Основатель и генеральный директор Overstock Патрик М. Бирн прокомментировал идею компании, сказав:

"Как и любая экономика, винодельческой промышленности трудно масштабирования своих посредников тяжелых систем параллельно с растущими требованиями растущего глобального рынка.

VinX шаги в токениза вина фьючерсы, позволяя вино энтузиастов знать, без сомнения, что бутылки они покупают заполнены подлинными винами будет позиционировать всю отрасль в качестве модели новой глобальной экономики, которая заменяет старые сети мальчиков с трения доверия с помощью технологий ".
