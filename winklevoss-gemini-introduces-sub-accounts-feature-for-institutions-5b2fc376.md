# 윙클보스의 쌍둥이 자리는 기관에 대한 하위 계정 기능을 소개합니다 ...

###### 2019-09-03 15:09

뉴욕에 본사를 둔 암호 화폐 교환 쌍둥이 자리는 하나의 마스터 계정에서 여러 계정을 생성 할 수 있도록 기관 고객을위한 하위 계정 기능을 도입했다.

Sept. 3 발표에서, 윙클보스 형제 주도 의 암호 교환은 기관 고객이 단일 마스터 계정으로 하위 계정의 무제한 수를 만들고 관리 할 수있는 새로운 기능을 공개했다.

## "가장 일반적으로 요청된 기능"

사용자가 하위 계정 세트를 만들면 거래 전략, 자금 및 최종 고객을 분리할 수 있으며 보다 간단한 방법으로 볼륨 기반 거래 수수료 할인을 받을 수 있습니다.

발표는 더 설명 :

"각 하위 계정은 특정 역할에 대한 권한 부여된 고유한 사용자 집합과 자체 계정 잔액 및 독립적으로 검증 가능한 고유암호화 주소를 가질 수 있습니다.

마스터 계정 관리자는 모든 하위 계정의 모든 사용자에 대해 작업을 보고 수행할 수 있습니다."

## 제미니의 서비스 확장

이 발표는 쌍둥이 자리가 피아트 전송 속도를 높이기 위해 암호 친화적 인 은행 실버 게이트의 실버 게이트 교환 네트워크에 합류했다는 뉴스의 여파로 온다.

Silvergate와의 거래소 파트너십의 주요 이점은 쌍둥이 자리의 기관 투자자 고객이 언제든지 계좌에서 미국 달러를 인출하고 입금 할 수 있다는 것입니다.
