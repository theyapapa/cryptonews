# Потоковая платформа для геймеров начинает публичное бета-тестирование собственной цепочки...

19 сентября, плай2ливе потоковая платформа для геймеров и любителей Кибера официально объявила о запуске открытого бета-тестирования их цепочки потокового потока для широкой публики.

Новая платформа для создания цепочки блоков предназначена для того, чтобы позволить участникам воспользоваться схемами монетизации, включая растяжки, зрителей и организаторов турниров.

# # сбор геймеров и влиятельных как компания сообщила коинтелеграф, 15 августа, доступ к фазе тестирования был впервые предоставлен представителям средств массовой информации и аккредитованных растяжки для того, чтобы проверить эксплуатационные возможности платформы основных Функциональность.

По словам Генерального директора плай2ливе Алексея Бурдыко, некоторые разработки новой платформы не использовались раньше: «по мнению ведущих экспертов, одним из интересных преимуществ проекта является возможность покупки и заработка Люк маркеры непосредственно \ [на \] веб-сайт платформы ", он прокомментировал в официальном пресс-релизе.

Для ускорения доставки контента от растяжки до зрителя создатели плай2ливе реализовали на платформе технологию сети доставки контента (CDN).

# # цепочки безопасности вопрос в соответствии с плай2ливе технический директор Владислав Арбатов, потоковая платформа стала безопаснее с цепочки.

"таким образом, плай2ливе будет первой платформой в мире, которая использует компактные устройства для обеспечения защиты информации для пользователей и обеспечения безопасности их идентификации", комментирует Арбатов.
