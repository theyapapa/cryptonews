# Запуск решить проблемы прослеживаемости в морепродуктах промышленности через цепочки...

###### 2018-09-03

На основе цепочки запуска фишкоин планирует применить децентрализованную технологию Леджер (DLT) к цепочкам поставок морепродуктов, связывая рыбаков и фермеров рыбы в точке сбора урожая для глобальных участников цепочки поставок морепродуктов, начиная от дистрибьюторов, переработчиков, оптовики и розничные торговцы.

Компания стремится к применению технологии цепочки, чтобы обеспечить прослеживаемость для индустрии морепродуктов и создать экосистему, которая могла бы позволить производителям морепродуктов и посредникам цепочки поставок получить вознаграждение через микротранзакции.

Начиная с 2017, еачмиле переосмысление мфиш как инструмента для мелких рыбаков и фермеров, которые делятся данными с сетью фишкоин.

# # отраслевой стимул модель фишкоин создает механизм вознаграждения производителей морепродуктов и посредников цепочки поставок посредством микротранзакций — с использованием фишкоин жетонов.

Несмотря на то, что название может предложить, маркеры не валюта для торговли морепродуктами, но "механизм для сбора и передачи данных стимулируя в различных формах, начиная с ключевых элементов данных (КДЕС), захваченных рыбаков и фермеров рыбы для целей прослеживаемости»,-говорится в техническом документе.

Проектная группа утверждает, что токены фишкоин могут служить как стимулом, так и механизмом сбора данных по цепочкам поставок морепродуктов.
