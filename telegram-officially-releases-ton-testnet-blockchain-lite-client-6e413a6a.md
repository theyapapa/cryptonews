# 전보 공식적으로 출시 톤 Testnet 블록 체인 라이트 클라이언트 ...

###### 2019-05-29 19:05

암호화 된 인스턴트 메시징 서비스 전보 공식적으로 그것의 전보 오픈 네트워크에 대 한 테스트 클라이언트를 발표 했다 (톤), test.ton.org에 게시 된 파일에 따르면.

다운로드 가능한 파일의 새로 도입 된 아카이브는 readme 정보 파일에 따라 톤 블록 체인 라이브러리의 관련 부분과 함께 톤 블록 체인 네트워크 라이트 클라이언트의 예비 버전의 분포 이다.

GitHub의 readme 파일에 따라, 클라이언트는 톤 블록 체인 테스트 넷에 대 한 전체 노드에 연결 하 여이에 대 한 몇 가지 쿼리를 전송-제대로 구성 된 경우.

라이트 클라이언트 자체 외에도 사용자는 라이트 클라이언트를 특수 서버에 연결 하는 스마트 계약 개발을 위한 구성 파일을 설치할 수 있습니다.

사용자는 Fift를 배포 하 여 스마트 계약을 로컬로 컴파일, 실행 및 디버그할 수 있으며 파일을 읽습니다.

이 언어는 TON 블록 체인 스마트 계약을 개발 및 관리 하 고 TON 가상 머신과 상호 작용 하기 위해 특별히 설계 되었습니다.

텔 레 그램은 4 월에 제한 된 수의 글로벌 개발자 들에 게 TON 블록 체인의 비공개 베타 테스트를 시작 했습니다.
