# Телеграмма официально выпускает TON Тестнет Blockchain Lite клиент ...

###### 2019-05-29 19:05

Зашифрованный сервис мгновенного обмена сообщениями Телеграмма официально выпустила тестового клиента для его Телеграмма открытая сеть (TON), согласно файлам, размещенным на test.ton.org.

Недавно введенный Архив файлов, доступных для скачивания, представляет собой дистрибутив предварительной версии TON Blockchain сети Lite клиента вместе с соответствующими частями библиотеки TON Blockchain, согласно информации из файла "ридме".

За файл, прочитанные на GitHub, клиент-если настроен правильно-подключен к полному узлу для тестовой сети blockchain TON и послал несколько запросов к нему.

Помимо самого Lite клиента, пользователи также могут установить файл конфигурации для разработки смарт-контрактов, который соединяет облегченный клиент с специальным сервером.

Пользователи также могут развернуть fift для компиляции, выполнения и отладки смарт-контрактов локально, файл читает.

Этот язык специально разработан для разработки и управления смарт-контрактами TON, и взаимодействует с виртуальной машиной TON.

Телеграмма, как сообщается, запустила частное бета-тестирование блокчейна TON для ограниченного числа глобальных разработчиков в апреле.
