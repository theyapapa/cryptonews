# Walmart's Latest Patent Unveils Blockchain-Based Delivery Management System

EE. UU. La más reciente aplicación de patentes relacionadas con el gigante minorista Walmart  [🔗](https://cointelegraph.com/tags/walmart) describe un sistema de administración de entrega "seguro", según un documento publicado por Estados Unidos. Oficina de Patentes y Marcas (USPTO) el 5 de julio.

La nueva aplicación, titulada "Delivery Reservation Apparatus and Method", describe un sistema de gestión basado en la cadena de ventas para los armadores de entrega que pueden mantener seguros los artículos entregados hasta que sus compradores puedan firmar y recolocarlos.

Un sistema de "reservas de bloque" es propuesto para "algunas modalidades" de un sistema para rastrear qué taquillas son libres, y que están ocupadas: "Cada espacio en la estacion de acoplamiento tiene una unidad de capacidad correspondiente para cada ubicacion en la estacion de acoplamiento. Las transacciones para las unidades de capacidad se rastrean en un libro mayor, con unidades de capacidad disponibles que indican una ubicación abierta en la estación de acoplamiento o unidades de capacidad de salida que indican que la ubicación tiene un localizador asegurado al mismo o que la ubicación está reservada para una entrega futura. "

 De acuerdo con la patente, "cada estación de acoplamiento puede ser un nodo dentro de una red de bloques", haciendo frecuentes referencias a un "libro mayor público" que actuaría como un registro abierto de las unidades disponibles y reservadas dentro del sistema.

A principios de esta primavera, Cointelegraph reportó  [🔗](https://cointelegraph.com/news/walmart-to-implement-blockchain-based-delivery-system)  sobre una solicitud de patente de Walmart para un "paquete inteligente" que bloquearía una serie de datos, incluyendo el contenido del paquete, las condiciones ambientales, la ubicación y otros detalles.

El gigante minorista ha estado desarrollando de forma proactiva una gama de soluciones de cadena de éxito para su negocio, que recientemente se asoció  [🔗](https://cointelegraph.com/news/walmart-ibm-blockchain-initiative-aims-to-track-global-food-supply-chain)  con IBM  [🔗](https://cointelegraph.com/tags/ibm)  para crear un sistema de cadena de bloques para el seguimiento de los alimentos a nivel global a través de su cadena de suministro.

En abril, Walmart anunció que estaba listo para usar la cadena de ventas TagPreserver,LNK7,63,71 para su negocio de alimentos en vivo, y la compañía ha presentado más patentes de bloqueo para los registros médicos  [🔗](https://cointelegraph.com/news/walmart-awarded-patent-for-blockchain-based-medical-records-system) , otorgados en junio, un mercado de reventa basado en la cadena de ventas  [🔗](https://cointelegraph.com/news/walmart-files-patent-for-blockchain-based-customer-marketplace), y un sistema de gestión de consumo de energía basado en criptografía.
