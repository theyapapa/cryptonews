# 러시아 국가는 공공 행정을 위한 블록 체인 기반 응용 프로그램을 테스트 ...

###### 2019-05-28 15:05

니즈니노브고로드의 러시아 국가는 5 월 28 일 ForkLog의 보고서에 따르면, 공공 행정 업무를 위한 블록 체인 기반 응용 프로그램 "도시 N"의 사용을 테스트 하 고 있습니다.

주 거주자는 도시 N 응용 프로그램을 사용 하 여 블록 체인 기반 보스 토크 플랫폼에서 세금을 납부 할 수 있을 것으로 알려졌다.

응용 프로그램의 다른 서비스는 니즈니노브고로드 주민의 신원을 확인 하는 능력을 포함, 뿐만 아니라 지출을 추적 할 수 있는 것은 공공 서비스 Gosuslugi의 온라인 포털을 통해 국가 예산에 지불 했다.

니 키 틴는 블록 체인 기술을 사용 하 여 투명성에 대 한 논평 뿐만 아니라 새로운 응용 프로그램을 통해 시민 참여에 대 한 정부의 희망을 주석, 니즈니노브고로드 지역의 주지사:

"새로운 기술 덕분에 주민들은 적극적인 ' 참여 ' 예산 책정을 통해 도시를 관리 하는 데 더욱 적극적으로 참여할 것입니다.

Blockchain 기술은 사람과 국가 사이의 관계에서 최대 투명성과 정직을 보장 할 수 있습니다.

Rostec 주 공사의 정보화에 대 한 국립 센터는 보도 전국 플랫폼의 구현을 감독 한다.

ForkLog 보고서에 따르면 Rostec와 보스 토크는 블록 체인 기술을 러시아 연방의 디지털 인프라에 구현 하기 위한 파트너십 계약을 체결 했습니다.
