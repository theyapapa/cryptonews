# Отчет о том, что Microsoft добавит блокчейн-инструменты в свою энергетическую платформу ...

###### 2019-06-10 10:06

Американский технологический гигант Microsoft, как сообщается, сделает инструменты на основе блокчейна доступными для пользователей PowerApps и Microsoft Flow, сообщило 10 июня новостное агентство GeekWire.

Microsoft якобы объявит о своих планах добавить блокчейн-инструменты к своей энергетической платформе во время саммита Microsoft Business Applications в понедельник в Атланте.

В частности, искусственный интеллект (ИИ) и инструменты блокчейн будут включены в систему системы пользовательских приложений PowerApps и инструмент бизнес-аналитики PowerBI.

Сама power Platform представляет собой набор инструментов - Power BI, PowerApps и Microsoft Flow - предназначенных для совместной работы по созданию пользовательских приложений, автоматизации рабочих процессов для повышения производительности бизнеса и анализа данных для получения информации.

В мае компания представила новый набор разработки блокчейна Azure Blockchain для блокчейна Ethereum.

Набор инструментов был разработан, чтобы помочь разработчикам, которые строят приложения с помощью платформы Microsoft blockchain as-a-service, Azure Blockchain Service.

Как сообщалось в конце мая, нью-йоркская блокчейн-компания ConsenSys также выпустила новый набор для разработчиков Blockchain и DApp Developer Job Kit, чтобы помочь начинающим разработчикам блокчейна Ethereum выйти на рынок.
