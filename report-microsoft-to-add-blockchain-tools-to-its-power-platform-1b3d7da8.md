# 보고서: 파워 플랫폼에 블록 체인 도구를 추가 하는 마이크로소프트 ...

###### 2019-06-10 10:06

미국 기술 거 대 한 마이크로소프트는 보도 blockchain 기반 도구를 사용할 수 있도록 PowerApps 및 마이크로소프트 흐름 사용자, 기술 중심 뉴스 아울렛 GeekWire 6 월에 보고 된 10.

Microsoft는 애틀랜타의 월요일에 Microsoft 비즈니스 응용 프로그램 정상 회담에서 블록 체인 도구를 전원 플랫폼에 추가 하는 계획을 발표 소문 것입니다.

특히 인공 지능 (AI) 및 블록 체인 도구는 회사의 PowerApps 사용자 지정 응용 프로그램 작성기 및 PowerBI 비즈니스 인텔리전스 도구에 포함 됩니다.

전원 플랫폼 자체는 사용자 지정 앱을 구축 하 고 워크플로를 자동화 하 여 비즈니스 생산성을 개선 하 고 데이터를 분석 하는 데 필요한 공동 작업을 위해 설계 된 Power BI, PowerApps 및 Microsoft Flow의 도구 모음입니다.

5 월, 회사는 Ethereum 블록 체인을 위한 새로운 Azure 블록 체인 개발 키트를 도입 했습니다.

이 도구 키트는 Microsoft의 서비스 블록 체인 서비스인 Azure 블록 체인 서비스를 사용 하 여 앱을 빌드하는 개발자를 지원 하도록 설계 되었습니다.

5 월 말에 보고 된 바와 같이, 뉴욕에 기반을 둔 블록 체인 회사 ConsenSys는 새로운 블록 체인과 DApp 개발자 직업 키트를 출시 하 여 주목 받는 Ethereum 블록 체인 개발자가 시장에 진입 하도록 돕습니다.
