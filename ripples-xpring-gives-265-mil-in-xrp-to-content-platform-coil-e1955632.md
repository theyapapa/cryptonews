# Ripple в Xpring дает $ 265 Mil в XRP для содержания платформы катушки ...

###### 2019-08-15 12:08

Развитие инфраструктуры Ripple и инициатива XRP по внедрению Xpring объявила о предоставлении гранта в размере 1 миллиарда XRP (примерно 265 миллионов долларов) на платформу монетизации Сети Coil 15 августа.

Сообщается, что coil будет использовать эти средства для создания экосистемы создателей, разработчиков, компаний и некоммерческих организаций, которые используют XRP через открытый стандарт Web Monetization.

Предоставление создателям контроля над дистрибуцией и монетизацией

(Альт-Н)

Coil является платформой, которая обеспечивает альтернативный метод для создателей, чтобы монетизировать их содержание в Интернете.

Генеральный директор Coil Стефан Томас прокомментировал:

"Создатели хотят больше выбора и контроля над тем, как их содержание монетизируется и распространяется на веб-монетизации обеспечивает решение, которое является более справедливым, открытым и инклюзивным для создателей и поклонников по всему миру".

Кость шеи, подключенная к головной кости

(Альт-Н)

Поскольку поклонники подписки наслаждаются контентом, Coil использует открытый API под названием Web Monetization для передачи микроплатежей создателям в режиме реального времени.

Web Monetization построен на протоколе Interledger, созданном совместно генеральным директором Coil Стефаном Томасом.

Итан Борода, старший вице-президент Xpring, сказал:

"Реклама и сайт по сайту подписки модели готовы к нарушению ( и низкой стоимости и быстрых транзакций XRP делает его идеальным инструментом для coil предложить альтернативный метод монетизации и оказывают положительное влияние на создателей".
