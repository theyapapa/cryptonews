# Siemens рассматривает возможность использования технологии блокчейн для каршеринга ...

###### 2019-07-15 20:07

В частности, Siemens изучает возможность использования блокчейна в транспортной отрасли, говорится в отчете Forbes от 15 июля.

По словам главы отдела кибербезопасности и блокчейна Siemens По корпоративным технологиям Андреаса Вида, Siemens стремится включить технологии блокчейн в каршеринг через Siemens Mobility – одну из дочерних компаний Siemens.

Вид сказал, что он считает, что этот тип технологии может быть улучшена с помощью решения blockchain:

"Это не только неудобно для водителей, это неудобно для компаний, потому что заправка карты получить украдены и они продаются в Интернете. различные участники, которые полностью доверяют друг другу, именно там блокчейн может добавить ценность».

Siemens также рассматривает другие области в транспортном секторе для блокчейн-решений.

Компания Siemens 'Corporate Technology' Сообщается, что на конференции Bosch's 2019 Connected World была представлена возможная блокчейн-решение для «смарт-парковки на основе блокчейна».

Согласно отчету, Siemens также рассматривает случаи использования блокчейна для цепочек поставок и производства.

Как ранее сообщал Cointelegraph, Grand View Research предположила, что блокчейн является цифровой технологией, движущей рост рынка в секторе глобальных систем управления транспортом (TMS).
