# 블록 체인 게임의 가상 레이싱 카는 $11만 이상 판매 합니다. ...

###### 2019-05-28 20:05

알 수 없는 사용자가 게임의 웹 사이트에 대 한 정보에 따라, 판매 시 415.9 에테르 (ETH) 또는 $110600에 대 한 다가오는 블록 체인 기반 게임 F1 델타 시간에 대 한 가상 레이싱 자동차를 샀다.

가상 레이싱 카는 ' 1-1-1 '이 최초로 발행 되는 모델 이며, 경매 페이지가 더 이상 출시 되지 않을 것 이라는 약속을가지고 있습니다.

이 회사는 게임 뒤에 있는 Animoca 브랜드는 31 개의 입찰가를 받은 후 "09E282" 사용자에 게 암호 수집을 판매 하기 전에 4 일 동안 리 미 티 드 에디션 모델에 대 한 경매를 실행 했습니다.

3 월 말, Animoca 브랜드는 포뮬러 1과 협력 하 여 게임을 출시 하 고, 플레이어는 가상 F1 자동차와 레이스를 서로 상대로 수집 합니다.

게임의 공식 웹 사이트에 따르면, 특정 자동차 모델을 소유 하는 게임에서 경쟁력 있는 이점이 있다:

"이 게임은 ERC-721 토큰 표준에 따라 대체 불가능 한 토큰 (NFTs)으로 존재 하는 고유 자동차, 드라이버 및 구성 요소의 수집과 거래를 중심으로 할 것입니다.

암호화 수집품은 기업이 블록 체인 기술을 사용 하 여 가상 수집품을 추적 하 고 인증 하는 비교적 최근의 혁신입니다.

2017의 끝 근처에, 인기 있는이 더 리 움 기반 게임 CryptoKitties는 그 수집품의 $1200만 이상 판매 했다, 이상으로가는 게임에서 가장 비싼 항목 $12만.
