# Виргинские Bitcoin - Большинство в спросе Крипта, который регулируется по-разному? ...

###### 2019-07-20 08:07

В мире, где глобальное криптосообщество продолжает сталкиваться с растущим числом регуляторных препятствий с каждым днем, термин «девственный Bitcoin» начинает становиться все более распространенным среди любителей цифровой валюты.

По словам Дэйва Джеванса(Dave Jevans), генерального директора CipherTrace, virgin Bitcoins по существу являются токенами BTC, которые не имеют записи транзакции (TX), связанной с ними.

Кроме того, блокчейн Bitcoin служит децентрализованным гекшем, который позволяет любому человеку следить за историей транзакций определенного токена одним нажатием кнопки.

Например, каждый биткойн несет с собой криптографически доказуемую историю, которая содержит подробную запись данных о собственности и транзакциях, связанных с токеном.

Учитывая всю вышеупомянутую информацию, важно учитывать, что на недавно завершившейся встрече «Большой двадцатки», которая состоялась в Осаке, Япония, основной руководящий комитет согласился принять стандарты Целевой группы по финансовым действиям (ФАТФ), которые обычно используется по отношению к фиатным валютам – даже для цифровых активов.

Однако, поскольку Bitcoin предлагает своим владельцам большую прозрачность по сравнению с твердыми деньгами, девственные монеты могут быть окрашены в чистоту в гораздо более простой манере.
