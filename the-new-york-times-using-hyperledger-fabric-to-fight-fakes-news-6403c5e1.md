# New York Times Использование Hyperledger Ткань для борьбы с fakes Новости ...

###### 2019-07-23 23:07

Документ записи, The New York Times (NYT), использует Hyperledger Ткань разрешенных blockchain, чтобы гарантировать происхождение цифровых файлов.

Согласно новому веб-сайту для NYT News Provenance Project, газета будет использовать технологию блокчейн для борьбы с дезинформацией в СМИ.

Технология блокчейн якобы поможет проекту в записи метаданных о видео- и аудиозаписях, которые публикуются новостными организациями.

Проект «Новый провансаль» в настоящее время находится на первом этапе, в котором он направлен на создание фотожурналистики ориентированных доказательство концепции, которая демонстрирует, как такая система на основе blockchain может работать для масштабирования.

На сайте говорится:

«Наша первоначальная работа включает изучение системы на основе блокчейна для записи и обмена метаданными о средствах массовой информации, в частности изображениями и видео, которые публикуются новостными организациями.

NYT отмечает: "Вся шумиха в сторону, blockchain предлагает механизмы для обмена информацией между субъектами таким образом, мы думаем, имеют важное значение для создания и поддержания происхождения цифровых файлов".

Роль, озаглавленная "Ведущий, Blockchain Exploration" искал,"перспективный лидер, который поможет представить и разработать на основе blockchain доказательство концепции для издателей новостей".
