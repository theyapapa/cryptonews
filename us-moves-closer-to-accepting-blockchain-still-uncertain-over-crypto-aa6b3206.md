# США движется ближе к принятию Blockchain, по-прежнему неопределенным над Crypto ...

###### 2019-07-27 17:07

Несмотря на опасения, высказанные многими регулирующими органами Соединенных Штатов в ходе недавних слушаний Комитета Палаты представителей по финансовым услугам о предлагаемой цифровой валюте Facebook, Весы, все большее число американских законодателей выступают в пользу разработки разрешительный регулятивный аппарат для предотвращения удушающей инноваций в криптовалютном секторе.

За несколько дней до слушаний принятие Закона о поощрении блокчейна свидетельствует о намерении правительства разработать разумную законодательную базу для контроля за возможным внедрением технологии блокчейн в различных государственных секторах.

Оделл далее сказал об этом:

"Принятие Закона о поощрении блокчейна Сенатом создает возможность для продвижения технологии на федеральном уровне и подвергать больше правительственных организаций тому, как это может принести больше прозрачности и эффективности в их работу.

Мацуи выразила предположение, что Закон о поощрении блокчейна заложит основу для содействия инновациям в секторе технологий распределенных реестров, заявив:

"Этот двухпартийный, двухпалатный законопроект объединит широкую группу заинтересованных сторон для разработки общего определения блокчейна и, возможно, что еще более важно, будет рекомендовать возможности для использования технологии для продвижения новых инноваций".

В Министерстве торговли США будет создана рабочая группа по блокчейну 

(Альт-Н)

В течение 90 дней после принятия Закона о поощрении блокчейна министр торговли США Уилбур Росс должен будет создать рабочую группу в министерстве торговли.

Группа, называемая Blockchain Рабочая группа, будет частично состоять из членов, представляющих "перекрестный раздел" федеральных учреждений, которые могли бы извлечь выгоду из принятия технологии распределенных реестров, и представители отраслевых заинтересованных сторон, академические учреждения, некоммерческие организации и группы по защите прав потребителей занимаются деятельностью, связанной с технологией блокчейн.

Доклад будет включать рекомендацию по определению "технологии блокчейн", а также рекомендации по изучению спектра потенциальных приложений для технологии распределенных реестров и предложения возможных приложений для технология блокчейн, которая будет способствовать повышению эффективности в федеральных агентствах США.
