# Состояние фермы тесты Блокчейн решение для ускорения авто страховые претензии ...

###### 2018-12-10 17:12

Американская страховая компания State Farm тестирует блокчейн-решение, чтобы ускорить процесс суброгации для автоматического утверждения, согласно сообщению, опубликованному 10 декабря.

Государственная ферма является большой группой страховых и финансовых услуг фирмы, которая обеспечивает Автострахование в США.

Государственная ферма работает над блокчейн-решением для ускорения суброгации процессов в страховой отрасли.

Суброгации является законным правом, проводимым большинством страховых перевозчиков легально преследовать третью сторону, которая привела к потере страховки застрахованным, и, как правило, последняя часть процесса страховых претензий.

Сообщается, что государственная ферма тестирует блокчейн-решение совместно с другим страховщиком, чтобы узнать, может ли он сократить время, необходимое для завершения процесса суброгации, автоматически собирая все суброгации суммы платежей.

По словам Майка поля, Исполнительный инновации государственного хозяйства, "суброгации является относительно ручной, трудоемкий процесс часто требующих физических проверок, которые будут отправлены на претензии по претензии основе между страховщиками".

В сентябре, крупная страховая компания народной страховой компании Китая (PICC) в партнерстве с блокчейн-платформой Вечаин и глобальной компанией по контролю качества и управлению рисками, чтобы сделать свой бизнес более эффективным и экономичным.
