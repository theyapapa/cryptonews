# 에테 리움 기반 의 거리 향상 애플 리케이션 베일 은 가게를 닫습니다 ...

###### 2019-07-12 19:07

7 월 11 일 공식 매체 게시물에 따르면 이더리움 기반 예측 플랫폼 베일이 종료됩니다.

베일의 공동 창업자 인 폴 플레처 힐 (Paul Fletcher-Hill)은 사용자가 오픈 포지션을 구속하고, 활성 시장에서 포지션을 철회하고, 베일 이더를 철회하고 이더로 전환할 것을 권장했습니다.

베일은 이더리움 기반 예측 시장 오구르에 대한 확장의 일종이었다.

Augur는 예측 시장입니다 - 여전히 존재 - 사용자가 만들고 암호 화폐 이더와 이벤트의 결과에 내기 할 수 있도록 스마트 계약을 사용합니다.

예를 들어, 언론 보도 에서 오구르 시장에 나열된 상위 3 개의 베팅은 "노박 조코비치가 2019 윔블던 남자 단식 우승자가 될 것인가?

웹 사이트에 따르면 베일은 거래 프로세스를 가속화하여 "Augur 주류를 가져 와서 사용자 경험을 향상시키기위한 것"이었습니다.

베일은 사용자가 0x 프로토콜을 통해 Augur 시장에서 더 빨리 거래할 수 있게 하고, 사용자가 블록 체인에서 Augur 거래를 네이티브로 완료하기 전에 베일에 주식을 판매 할 수 있도록하여 즉각적인 정산을 제공했습니다.
