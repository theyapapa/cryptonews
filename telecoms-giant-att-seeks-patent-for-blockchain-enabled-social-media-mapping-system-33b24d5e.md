# telecoms 자이언트 AT&T 블록 체인 기반 소셜 미디어 ' 매핑 ' 시스템에 대 한 특허를 찾고 있습니다. ...

###### 2018-12-14 22:12

미국의 통신 자이언트 AT&T는 블록 체인 기반의 소셜 미디어 역사 "지도"에 대 한 특허를 찾고 있습니다.

AT&T의 특허 출원은 다양 한 목적으로 사용 될 수 있는 가입자의 데이터를 저장 하는 거래 이력 제어기를 포함할 수 있는 블록 체인 기반 시스템을 설명 한다.

이 문서에는 "여기에 설명 된 소셜 미디어 기록 맵 플랫폼은 블록 체인 레코드의 불변 하 고 영구적인 특성을 활용 하 여 여러 곳에서 발생 하는 온라인 트랜잭션을 나타내는 데이터를 저장 하 고 액세스를 제공할 수 있습니다. 소셜 미디어 응용 프로그램. "

제출에 따라 콘텐츠 제작자는 "매핑" 플랫폼에서 데이터의 소유권을 유지할 것입니다. 그러나 사용자 간에 블록 또는 데이터의 소유권을 전달 하는 대신 소셜 미디어 계정 소유자는 자신의 온라인 거래 데이터에 대 한 주 소유권을 유지 합니다.

소셜 미디어 계정 소유자의 추종자와 같은 소셜 미디어 기록 맵 서비스의 다른 사용자에 게 소셜 미디어 계정 소유자가 전달 하는 것은 높은 가시성 권한의 개념입니다. "

또한 그 달, 금융 서비스 거 대 아메리칸 익스프레스는 영수증의 이미지를 캡처하고 전송 하는 블록 체인 기반 시스템에 대 한 특허를 출원 했다.

시스템은 "광학 문자 인식"을 통해 이미지를 해독 하 고이를 "관련 레코드", 즉 거래 기록과 일치 시킵니다.
