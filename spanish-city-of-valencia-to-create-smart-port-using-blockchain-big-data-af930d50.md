# 발렌시아 스페인어 도시 블록 체인을 사용 하 여 ' 스마트 포트 '를 만들려면 빅 데이터 ...

###### 2018-10-04 15:10

발렌시아 시에 기반을 둔 스페인에서 가장 분주 한 항구 중 하나는 블록 체인과 빅 데이터 기술을 사용 하는 "스마트 포트"의 생성을 발표 했습니다.

발렌시아 항만 당국에서 신기술을 구현 하는 책임을 맡고 있는 호세 가르시아 드 라 기아는 발렌시아에서 뿐만 아니라 많은 국제 항만에서의 물류 개선을 위한 좋은 옵션으로 블록 체인을 볼 수 있다고 설명 하 고 있습니다: "발렌시아에서 출발, 우리는 블록 체인을 전략적 옵션으로 사용 하 여 물류 체인의 투명성을 제공 하며,이는 당사 항만 보다 더 나아가 게 됩니다.

또한 블록 체인은 전 세계의 자원을 최적화 하 고 유지 보수에 소요 되는 시간을 단축 하 고 비용을 절감할 수 있도록 "서류 없이 포트"를 만드는 데 도움이 될 것 이라고 밝혔다.

메시지는 스마트 포트 동안 전달 되었다 & 서플라이 체인 테크놀로지 컨퍼런스 네덜란드 도시 로테르 담에서 개최 이번 주,이는 발렌시아의 포트에서 관계자 뿐만 아니라 앤트 워프 (벨기에)와 알 헤 시 라스 (스페인)의 대표자 들이 참여 했다.

블록 체인은 물류를 개선 하기 위해 국제 항만에서 널리 사용 됩니다.

예를 들어, 영국의 주요 항만 운영 회사 (ABP)는 분산형 솔루션을 사용 하는 파일럿 발송물에 곧 참여 하 고, 덴마크는 지역 선박 등록을 위해 블록 체인을 구현할 계획을 발표 했습니다.

그러는 동안 아부다비 항구의 자회사는 국내 물류 블록 체인 네트워크를 시작 했습니다.
