# Испанский город Валенсия для создания "смарт-порт" с помощью цепочки, большие данные ...

###### 2018-10-04 15:10

Один из самых оживленных портов Испании, базирующихся в городе Валенсия объявил о создании "смарт-порт", который будет использовать цепочки и технологии больших данных.

Хосе Гарсия де ла Гия, который отвечает за внедрение новых технологий в портовом органе Валенсии, объясняет, что они видят, что цепочки являются хорошим вариантом для улучшения логистики не только в Валенсии, но и во многих международных портах: "начиная с Валенсии, Мы предлагаем использовать цепочки в качестве стратегического варианта для обеспечения прозрачности логистической цепочки, от конца до конца, идя дальше, чем наш порт.

Де ла Гия также говорит, что цепочки поможет создать "порты без документов", чтобы оптимизировать ресурсы по всему миру, чтобы сократить время, затрачиваемое на техническое обслуживание, а также сократить расходы.

Сообщение было доставлено во время смарт-портов и цепи поставок технологий конференции, состоявшейся в голландском городе Роттердам на этой неделе, в котором чиновники из порта Валенсии, а также представители из Антверпена (Бельгия) и Альхесирас (Испания) приняли участие.

Цепочки широко используются международными портами для улучшения логистики.

Например, ведущий портовый оператор Великобритании, ассоциированные британские порты (адресной книги), в скором времени примут участие в пилотных поставках с использованием децентрализованных решений, в то время как Дания выявила свои планы по внедрению блоков для локальных судовых регистров.

Тем временем, дочерняя компания в портах Абу-Даби также запустила внутреннюю логистическую сеть.
