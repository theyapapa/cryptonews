# США Национальный свиной Совет для пилота Blockchain Tech после нового партнерства ...

###### 2019-03-18 16:03

Национальный совет по свинине Соединенных Штатов в партнерстве с блокчейн запуска ripe.io проверить блокчейн-платформу для свиней цепи поставок, в соответствии с пресс-релизе, опубликованном 18 марта.

Раджа Рамачандран, со-основатель ripe.io, показал, что "через блокчейн, клиенты, как Национальный совет свинины может позволить своей организации и членов для создания общих, неизменяемых надежных записей, которые касаются критических продовольственных вопросов, таких как устойчивость, качество, прослеживаемость, отходы и мошенничество».

Отходы от крупномасштабного свиноводства стала проблемой для жителей таких государств, как Айова и Северная Каролина, где промышленность имеет значительное присутствие.

Технология Блокчейн была развернута многими компаниями по всему миру, что позволило повысить эффективность цепочки поставок.

В последнее время в Северной Америке крупнейших фирменных шельфа стабильной морепродукты фирма Шмель Bee Foods запустила блокчейн платформу для морских прослеживаемости в партнерстве с немецкой технологии компании SAP.

Новая платформа может якобы контролировать цепочку поставок Желтоперый тунца из Индонезии к конечным потребителям.

CQL рекламируется, чтобы гарантировать потребителям полную прослеживаемость продукции по всей цепочке поставок.
