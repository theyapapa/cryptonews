# Стартап США поднимает $14,1 млн на Блокчейн-сеть платежей для ритейлеров ...

###### 2019-04-11 15:04

Нью-Йоркский блокчейн-стартап Flex поднял $14 100 000 на разработку сети платежей для ритейлеров.

В релизе, Flex поднял $14 100 000 в финансировании от таких участников, как на раннем этапе маркера фонда 1kx, инвестиционные фирмы доступ предприятий и Нима капитал, и хедж-фонд пантера капитал, который недавно показал, что он был близок к завершению финансирования для его Третий венчурный фонд, уже поднимая $160 000 000.

Компания намеревается создать платежную сеть для розничных торговцев, которая позволит снизить затраты, накладные расходы и мошенничества с помощью блокчейн-расчетов.

Flex также планирует выпустить мобильное приложение, с помощью которого клиенты могут проводить операции с криптовалюты они уже владеют.

Тайлер Spalding, соучредитель и Генеральный директор компании "флекса", сказал, что "анти-мошенничество и стоимость выгоды глобальных платежей криптовалют огромны, но есть много препятствий для интеграции принятия для торговцев и потребителей, так.

Ранее сегодня, Соединенные Штаты пищевой и лекарственной цепи Albertsons компании объявили, что будет использовать IBM продовольственной Trust платформы блокчейн для отслеживания цепочки поставок для салата Ромен, но стремится к разветвлению в другие продукты.

В прошлом месяце американский Совет по свинине в партнерстве с блокчейн-стартапом ripe.io для тестирования блокчейн-платформы для цепочек поставок свинины.
