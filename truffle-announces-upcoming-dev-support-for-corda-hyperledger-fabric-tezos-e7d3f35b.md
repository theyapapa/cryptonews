# 트뤼플, 하이퍼레저 패브릭, 테조스에 대한 곧 개발 지원 발표 ...

###### 2019-08-02 21:08

블록 체인 개발 회사 인 Truffle은 블록 체인 프로토콜 인 Corda, Hyperledger Fabric 및 Tezos에 대한 지원을 포함하도록 완전히 관리되는 개발자 제품군을 확장 할 계획을 발표했습니다.

트러플의 창립자이자 CEO인 팀 쿨터(Tim Coulter)는 워싱턴 주 레드먼드에 있는 마이크로소프트 캠퍼스에서 트러플콘 2019-트러플의 연례 개발 컨퍼런스에서 회사의 계획을 발표했다.

Cointelegraph와 공유 된 보도 자료에 따르면, 트러플 스위트는 스마트 계약에 대한 백 엔드 작업에서 응용 프로그램의 프론트 엔드 생성에 이르기까지 개발 워크 플로우를위한 완벽하게 관리되는 블록 체인입니다.

트러플의 개발자 도구는 ConsenSys의 벤처 프로덕션 스튜디오에서 개발된 2015년으로 거슬러 올라간다고 합니다.

프레스 타임에 트러플의 개발자 제품군은 이더리움과 쿼럼의 블록체인 프로토콜을 지원합니다.

Cointelegraph에 의해 이전에보고된 바와 같이, 교통 얼라이언스 (BiTA)의 운송 그룹 블록 체인은 블록 체인 기술을 통해 공급망 상호 운용성을 추진하려고합니다.

BiTA 회장 패트릭 더피는 그룹의 목표에 대해 언급하면서 블록 체인은 "화주와 운송 업체 간에 발생하는 거래를 원활하게 할 수있는 잠재력을 가지고 있지만 운송 리더의 적극적인 참여가 필요합니다."
