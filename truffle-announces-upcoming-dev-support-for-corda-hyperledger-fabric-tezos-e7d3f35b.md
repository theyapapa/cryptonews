# Трюфель объявляет о предстоящей поддержке Dev для Корда, Hyperledger Ткань, Tezos ...

###### 2019-08-02 21:08

Блокчейн-разработчик компании Truffle обнародовал планы по расширению своего полностью управляемого пакета разработчиков, чтобы включить поддержку блокчейн-протоколов Corda, Hyperledger Fabric и Tezos.

Основатель и генеральный директор Truffle Тим Коултер объявил о планах компании на Конференции Посевных ВКС в Редмонде, штат Вашингтон, на ежегодной конференции разработчиков Трюфеля.

Согласно пресс-релизу, совместно с Cointelegraph, Трюфель Suite является полностью управляемым блокчейном для рабочего процесса разработчиков, от бэкэнда работы по смарт-контрактам до создания приложений.

Сообщается, что инструменты для разработчиков Truffle датируются 2015 годом, когда они были разработаны в рамках венчурной студии ConsenSys.

В прессе пакет разработчиков Трюфеля поддерживает блокчейн-протоколы Ethereum и Кворума.

Как ранее сообщал Cointelegraph, транспортная группа Blockchain in Transport Alliance (BiTA) пытается обеспечить совместимость цепочки поставок с помощью технологии блокчейн.

Президент BiTA Патрик Даффи прокомментировал цели группы, отметив, что блокчейн «имеет потенциал для сглаживания транзакций, которые происходят между грузоотправителями и перевозчиками, но требует активного участия лидеров перевозок».
