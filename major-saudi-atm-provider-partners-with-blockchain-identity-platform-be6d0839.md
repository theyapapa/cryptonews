# 블록 체인 아이덴티티 플랫폼과 함께 하는 사우디아라비아의 주요 ATM 제공 업체 파트너 ...

###### 2019-03-01 19:03

사우디 자동 입출금 기 (ATM) 공급자 Alhamrani 유니버설 (AU)는 블록 체인 기반 생체 인식 ATM을 개발 하기 위해 2 월 27 일에 발표 된 보도 자료에 따라 blockchain id 플랫폼 shocard와 협력 하고있다.

컬 래 버 레이 션은 AU와 shocard가 5 단계 인증 프로세스의 일환으로 블록 체인 기술과 생체 인식을 결합 한 새로운 유형의 ATM을 만들 것입니다.

atm은 지역 내 여러 은행에서 통신할 수 소문, 블록 체인 기술을 구축 하면 은행 데이터베이스에 액세스할 필요 없이 사용자의 신원을 확인할 수 있습니다.

shocard의 설립자 이자 CEO 인에 브로 히 미는 블록 체인이 "개인의 정체성을 보호 하는 것"으로 은행이 철수를 합법적으로 확인 하는 데 필요한 정보를 확인 한다고 말했습니다.

블록 체인은 신원 관리 시스템을 위해 널리 채택 되 고 있습니다.

최근 글로벌 컴퓨터 기술 거 인 오라클은 블록 체인 거래 내역을 통합 하는 것과 같은 id 관리 및 데이터 통합 기능을 포함 하 여 상업용 Oracle 블록 체인 플랫폼에서 기능의 확장을 발표 했습니다. 기타 데이터 리소스.

지난해 11 월 미국 소프트웨어 회사 마이크로소프트는 오프 체인 id 및 데이터, 모니터링 및 메시징 응용 프로그래밍 인터페이스 (API)와 같은 기능을 포함 하는 서버 리스 블록 체인 기반 Azure 개발 키트를 출시 했습니다. 블록 체인 기반 앱을 개발 하는 데 사용할 수 있는 형식입니다.
