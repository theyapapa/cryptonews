# Телеграмма бить Facebook удар с Gram Token октября запуск ...

###### 2019-08-28 04:08

Зашифрованное приложение для обмена сообщениями Telegram запустит свою цифровую валюту, Gram, в октябре, инвесторы сообщили основные средства массовой информации.

Токены Telegram должны быть представлены через два месяца

(Альт-Н)

Как сообщила 27 августа газета New York Times, три анонимных инвестора в предложение частных первичных монет (ICO) на $1,7 млрд заявили, что компания подтвердила, что выпустит первые токены через два месяца.

Долгосрочная цель для руководителей состоит в том, чтобы сделать Gram доступным для всех примерно 300 миллионов пользователей Telegram, добавили инвесторы.

Если это правда, Telegram будет бить Facebook Весы в запуске общественного глобальной цифровой валюты.

Регуляторный контроль ждет

(Альт-Н)

Остается неизвестным, как Telegram планирует успокоить власти, обеспокоенные свободной альтернативой фиатной валюте, но Telegram уже сталкивался с проблемами из-за Gram.

Как сообщает Cointelegraph, компания отменила публичный этап ICO, в то время как несколько компаний стремятся продавать токены GRAM неофициально, иногда по завышенным ценам.

Telegram также бить другого лица заинтересованы в выпуске Весы конкурента, Народный банк Китая, или PBoC, который на этой неделе объявил развертывание для своей цифровой валюты начнется 11 ноября этого года.
