# Стартап под руководством ветеранов JPMorgan привносит бесплатную платформу BaaS в Azure ...

###### 2019-08-27 19:08

Стартап гибридных блокчейн-решений Kadena обновил свою корпоративную блокчейн-платформу на Microsoft Azure Marketplace.

27 августа Компания Kadena объявила, что ее новый продукт под названием Kadena Scalable Позволенный блокчейн доступен на Azure Marketplace бесплатно.

Амбиции масштабируемости блокчейна

(Альт-Н)

Основанная бывшими экспертами по блокчейну в американском банковском гиганте JPMorgan, Kadena впервые выпустила свой разрешенный блокчейн на Amazon Web Services Marketplace в январе 2019 года.

Частная цепочка для корпоративного использования

(Альт-Н)

Обновленная корпоративная блокчейн-платформа Kadena поддерживает до четырех узлов и 2000 транзакций в секунду, говорится в сообщении.

Уилл Мартино, соучредитель и генеральный директор Kadena, сказал TechCrunch, что недавний запуск их корпоративного блокчейн-продукта на Azure основывается на успехе прошлогодней работы с участием AWS.

И, выведя эту цепочку на рынок AWS, а теперь и на Microsoft Azure, мы обслуживаем почти весь корпоративный блокчейн-рынок, который происходит в облаке».

Соучредители Popejoy и Martino - оба из которых работали в JPMorgan на ранних блокчейн-проектах до запуска Kadena в 2016 году - Kadena привлекла $14,9 млн.
