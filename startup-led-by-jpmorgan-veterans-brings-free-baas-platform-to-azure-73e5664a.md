# JP모건 재향 군인이 이끄는 스타트업은 Azure에 무료 BaaS 플랫폼을 제공합니다. ...

###### 2019-08-27 19:08

하이브리드 블록 체인 솔루션 스타트업 인 Kadena는 Microsoft Azure 마켓 플레이스에서 엔터프라이즈 블록 체인 플랫폼을 업데이트했습니다.

8월 27일, Kadena는 자사의 신제품인 카데나 확장 가능한 인증 블록체인을 Azure 마켓플레이스에서 무료로 이용할 수 있다고 발표했다.

## 블록체인 확장성 야망

미국 의 금융 거대 기업 JPMorgan에서 전 블록 체인 전문가에 의해 설립 된 Kadena는 2019 년 1 월 아마존 웹 서비스 마켓 플레이스에서 허가 된 블록 체인을 처음으로 발표했습니다.

## 엔터프라이즈용 프라이빗 체인

Kadena의 업그레이드 된 엔터프라이즈 블록 체인 플랫폼은 발표에 따라 최대 4 개의 노드와 초당 2,000 개의 트랜잭션을 지원합니다.

카데나(Kadena)의 공동 창립자이자 CEO인 윌 마르티노(Will Martino)는 테크크런치(TechCrunch)와의 인터뷰에서 최근 Azure에서 엔터프라이즈 블록체인 제품이 출시된 것은 AWS와 관련된 작년의 노력의 성공에 기반을 둔 것이라고 말했다.

그리고 이 체인을 AWS 마켓플레이스에 도입하여 이제 Microsoft Azure에 서비스를 제공함으로써 클라우드에서 발생하는 거의 모든 엔터프라이즈 블록체인 시장에 서비스를 제공합니다."

2016년 카데나(Kadena)를 출시하기 전에 JPMorgan에서 초기 블록체인 프로젝트에 참여한 포프조이와 마르티노가 공동 설립한 카데나는 지금까지 1,490만 달러를 모금했다고 테크크런치 보고서는 밝혔다.
