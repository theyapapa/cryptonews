# США Блокчейн фирмы партнеры с устойчивой строительной фирмы для проектирования центров обработки данных ...

###### 2019-04-02 21:04

Базирующиеся в Соединенных Штатах Блокчейн Holdings Capital венчурные предприятия (БХКВ) сотрудничают с блоками SG, которые используют контейнеры для перевозки грузов в строительстве для разработки своих многоконтейнерных центров обработки данных, согласно пресс-релизу 1 апреля.

Блокчейн-компания, которая специализируется на недвижимости, энергетике и технологических компаниях, будет единственным партнером SG блоков в проектировании децентрализованных центров обработки данных.

БХКВ в конечном итоге планирует открыть канал дистрибуции для масштабирования производства ЦОД, из которых компания стремится производить 150 – 200 на 2020.

Большинство центров будет исправлено, но компания также будет развивать мобильные.

Например, мобильный центр обработки данных может быть помещен в отдаленные районы для того, чтобы сократить расходы на электроэнергию.

Блоки SG, основанные в 2007, разрабатывают устойчивые строительные решения с использованием контейнеров для перевозки грузов.

Он рассмотрит, как блокчейн может повысить эффективность и Управление сделками с недвижимостью и передачей земельных титулов по нескольким графствам.
