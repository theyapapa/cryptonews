# 에이 더 리 움 2.0는 ' 정말 멀지 않은 곳에 ' 출시 Devcon4 기조 연설에서 비 alik Buterin를 말한다 ...

###### 2018-11-01 08:11

이 더 리 움 공동 설립자 비 리 크 Buterin는 10 월 31 일 프라하의 Devcon4 회의 기조 발표에서 "일관 된 전체"이 더 리 움 2.0으로 묘사 된 로드맵을 발표 했습니다.

불리 한 ' 평온, ' 프로젝트는이 더 리 움 개발자가 2014부터 작업 한 여러 프로젝트를 포함 합니다.

"이 더 리 움 2.0은 \ [...] 우리가 몇 년 동안 이야기 한 다른 기능의 무리의 조합, 몇 년에 대 한 연구, 적극적으로 전체 하나의 화합에 함께와 서 몇 년 동안 구축," Buterin 말씀을 요약 한다.

에이 더 리 움 사용자와 분석가는 오랫동안 네트워크 업그레이드의 잠재적 인 기간을 논의 하 고 있으며,이는 기술적인 노하우를 강화 하기 위해 최근의 업데이트가 밀려 나는 것을 보았습니다.

여기에는이 더 리 움 2.0의 적어도 2 개의 구현을 통합 하는 testnets-안정화 프로토콜 사양 및 크로스 클라이언트 testnets이 포함 됩니다.

추가 시험은 중요 하다, 그는이 더 리 움 1.0 릴리스에서 배운 것을, 그가 제안 너무 빨리 일어났다.

이번 주에 Devcon4에서 연설 했을 때, erc-20 토큰 표준을 공동 집필 한이 더 리 움 개발자는 그가 주장 하는 초기 코인 제공 (ICO)을 위한 새로운 모델을 도입 하 여 사기 로부터 투자자를 더 잘 보호 하 게 될 것이 라 Buterin.
