# 블록 체인 기술을 사용 하 여 의료 데이터 권한을 관리 하는 새로운 Bitfury 합동 프로젝트 ...

###### 2019-04-04 19:04

Bitfury 그룹 및 블록 체인 기반 의료 데이터 마켓플레이스 론 제네시스는 의료 산업을 위한 블록 체인 기반의 동의 관리 시스템의 생산을 공동으로 시작 했습니다.

릴리스 당 신제품은 지속적이 고 예정 된 연구 노력에 대 한 사용자 동의 관리, GDPR (일반 데이터 보호 규정 준수) 및 건강 보험 이식성 및 책임 법 (HIPAA)에 대 한 문제를 해결 합니다. 뿐만 아니라 의료 연구를 위한 데이터 수집을 간소화.

수집 된 사용자 동의에 대 한 모든 데이터 및 업데이트는 향후 감사를 위해 타임 스탬프가 포함 된 ExonumTM 블록 체인에 기록 됩니다.

알렉스 자 보로 코 프의 CEO 인 인실 리코 의학 대표이사는 말했다: "규정은 국가 마다 다르지만 연구 조직은 데이터 소유자를 어떤 규정 보다도 훨씬 더 잘 보호 하는 콘텐츠 관리 시스템을 구축 하기 위해 노력 해야 합니다.

2 월에 Bitfury 방사선 블록 체인 마켓플레이스 의료 진단 웹 (MDW)과 협력 하 여 블록 체인 기반의 의료 영상 생태계를 구축 했습니다.

캘리포니아 대학에서 학자, 샌프란시스코, 또한 블록 체인 기반 시스템을 사용 하 여 의료 데이터를 공유 하는 방법을 제안 했다.

또한,이 시스템은 연구 중에 불리 한 사건을 보고 하 고 의료 기록 관리를 개선 하기 위한 방법을 발전시키는 것을 목표로 합니다.
