# Исследователи обнаружили угрозу "необычных" Виртуальная машина Crypto Mining ...

###### 2019-06-24 04:06

Фирма по кибербезопасности ESET обнаружила, что она описывает как необычную и настойчивую криовалютную майнер, распространяемую для macOS и Windows с августа 2018 года.

Согласно ESET, новая вредоносная программа, получившая название "LoudMiner", использует программное обеспечение для виртуализации - VirtualBox на Windows и ЗЕМУ на macOS - для добычи криптографии на виртуальной машине Tiny Core Linux, тем самым имея потенциал, чтобы заразить компьютеры через несколько операционных систем.

Сообщается, что сам майнер использует XMRig - программное обеспечение с открытым исходным кодом, используемое для майнинга altcoin monero (XMR) с открытым исходным кодом и пул майнинга, что якобы препятствует попыткам исследователей повторить транзакции.

Исследование показало, что как для macOS, так и для окон майнер работает в пиратских приложениях, которые объединяются с программным обеспечением виртуализации, изображением Linux и дополнительными файлами.

ESET отмечает, что майнер нацелен на приложения, цели которых связаны с аудиопроизводством, которые обычно работают на компьютерах с надежной вычислительной мощностью и где высокое потребление процессора - в данном случае вызванное стелс-крипто-майнингом - не может поразить пользователей как Подозрительных.

ESET определила три штамма майнера, нацеленных на системы macOS, и только один для Windows до сих пор.

Сетевые соединения с необычными доменными именами - из-за скриптов внутри виртуальной машины, которые связываются с сервером C'C для обновления конфигурации майнера - это еще одна поддавки, добавляют исследователи.
