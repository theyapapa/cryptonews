# 연구진은 트위터에 15k는 암호 관련 사기 봇의 네트워크를 공개 ...

오늘 발표 된 새로운 연구, 8 월 6 일, 암호 화폐의 악명 높은 현상에 빛을 흘리 고 있다-가짜 "경품을 광고 하는 트위터 계정," 적어도 15000 사기 봇의 네트워크를 공개.

트위터 봇의 분석,이는 악화를 계속 하 고 심지어 순진한 사용자를 속일, 사이버 보안 회사 듀오 시큐리티에서 왔어.

이 프로젝트는 8800만 트위터 계정의 거 대 한 tranche 참여, 연구자 로봇 분류자를 훈련 하는 기계 학습 기술을 사용 하 여.

각 계정의 최신 200 짹짹을 사용 하 여, 분류자는 가짜 대회를 확산 작업에서 15000 봇의 메쉬를 발굴 하 고 암호화 화폐 업계의 가장 잘 알려진 인물과 기업의 일부를 사칭.

그 적극적으로 "암호화 트위터에 대 한," 봇이 장면의 거의 일부가 되 고, 그들의 가짜 판촉의 편 재로 인해.

"봇 ' 탐지를 막기 위해 시도는 콘텐츠 주위의 메타 데이터를 포함 하 여, 전체적으로 계정 분석의 중요성을 보여," 아니 스는 계속: "예를 들어, 봇 계정은 일반적으로 짧은 버스트에서 짹짹 것입니다, 평균 시간을 일으키는 짹짹 사이에 매우 낮은 것입니다.

대부분의 경우 자동 탐지를 기반으로 스팸 성 콘텐츠가 트위터에 숨겨져 있으며, 트위터 계정의 5% 보다 "방어에 추가" \ [l \] ess는 스팸 관련입니다. "
