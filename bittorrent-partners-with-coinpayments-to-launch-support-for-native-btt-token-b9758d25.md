# BitTorrent-партнеры с Коинпайментс для запуска поддержки собственного токена BTT ...

###### 2019-02-07 10:02

Одноранговый (P2P) торрент-клиент BitTorrent объявил, что его родной токен, BitTorrent Token (BTT), основанный на протоколе трона, теперь поддерживается шлюзом крипто-платежей Коинпайментс.

Как сообщалось, BitTorrent запустила свой BTT токен в январе этого года-его первый полноценный шаг, чтобы начать разметить его содержание обмена экосистемы, примерно через шесть месяцев после того, как компания была приобретена децентрализованной интернет-стартапа трон.

По данным сегодняшнего пресс-релиза, BitTorrent уже сотрудничает с Коинпайментс по поддержке трона (TRX), Bitcoin (BTC) и Binance Coin (BNB) платежей за премиум услуги.

С добавлением поддержки BTT, пользователи теперь могут, как сообщается, использовать токен для оплаты BitTorrent и µ Torrent Pro подписки, которые предлагают преобразование файлов, HD качество потоковой передачи и защиты от вредоносных программ.

Джастин Сун, основатель трона и Генеральный директор BitTorrent, уточнил дорожную карту для разметки экосистемы BitTorrent, заявив, что партнерство с Коинпайментс является попыткой "сделать BitTorrent (BTT) жидкость в Интернет-рынке до \ [токен \] является полностью интегрированы в наши настольные и мобильные продукты».

Согласно BitTorrent сайт, приложение позволит пользователям обмениваться BTT для посева и пропускной способности для поддержки более быстрой загрузки раз.

Как сообщалось, «дорожная карта» для взаимной интеграции «трон» и «BitTorrent» была названа «Project Atlas», целью которой является создание децентрализованной платформы распространения контента с использованием криптовалют.
