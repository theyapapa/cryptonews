# «Cash» выкатывает нового программного обеспечения для укрепления здоровья сети ...

###### 2019-06-17 15:06

Некоммерческая организация Фонд «Кэш» в партнерстве с блокчейн-компанией Parity Technologies выпустила нового клиента с открытым исходным кодом для криптовалюты scash, говорится в пресс-релизе от 17 июня.

Новый клиент программного обеспечения, Зебра, якобы существует, чтобы обеспечить избыточность в случае своего первого клиента - scashd - не удается.

Кроме того, клиент предназначен для повышения производительности наличности другими способами, например, для того, чтобы иметь средства для «обнаружения ошибок, связанных с реализацией конкретных результатов», и предотвращения проблем, связанных с его механизмом консенсуса.

Parity Technologies построила других заметных клиентов криптовалюты в прошлом, таких как Parity Ethereum.

Паритет cTO Фредрик Гаррисон прокомментировал предполагаемые преимущества своего последнего клиента для scash, сказав:

«Сообщество выигрывает по всем направлениям, и теперь «Cash» может похвастаться более диверсифицированным сообществом, которое может эффективно адаптировать опыт как для разработчиков уровня 2, так и для конечных пользователей».

Как ранее сообщал Cointelegraph, криптовалютная фирма, стоящая за «Экэшом», в настоящее время сталкивается с судебным иском на 2 миллиона долларов, обслуживаемым недовольным экс-сотрудником.

Инженер-программист Саймон Лю подал в суд на бывшего работодателя компании «Зерокоин Электрик Монета» на том основании, что его контракт дал ему право на 15 000 единиц акций, а также на «Вознаграждение основателя», за которое компания сознательно не могла заплатить, согласно копии официального гражданского дела Подачи.
