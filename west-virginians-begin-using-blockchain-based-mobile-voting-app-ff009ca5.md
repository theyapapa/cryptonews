# West Вирджинии начать использовать на основе цепочки...

West Вирджинии начали использовать на основе цепочки для голосования мобильных приложений для литья заочного голосования в предстоящих среднесрочных выборов, шифер сообщил 25 сентября.

Граждане Западной Вирджинии, которые в настоящее время живут за рубежом, как сообщается, начали использовать цепочки с поддержкой приложений для голосования в пятницу, 21 сентября.

Пилотный проект для дистанционного голосования, который был доступен только для избранной группы избирателей, начался в марте и был успешно завершен 8 мая, в день первичных выборов в Западной Вирджинии.

В начале августа, Мак Уорнер, Западная Вирджиния государственный секретарь и воатз сообщил CNN об успешном завершении тестирования после "четырех проверок различных компонентов" платформы.

После доклада, Warner заместитель начальника штаба Майкл л. Королева сказала, что каждый отдельный округ Западная Вирджиния будет принимать окончательное решение об использовании приложения для ноябрьских выборов, добавив, что избиратели будут по-прежнему разрешается литья бумажных бюллетеней, если они выбирают.

Брэдли Туск из клыка Монтгомери филантропия — компания, которая финансировала разработку приложения, — поощряла развертывание цепочки для голосования.

Туск заявил, что дистанционное голосование может оказаться более избирателей, и в результате, "демократия будет работать намного лучше".
