# Binance API는 겉보기에 마진 거래가 9 개의 거래 쌍에 대해 활성화 되었다는 것을 보여 줍니다. ...

###### 2019-05-03 12:05

주요 암호 화폐 거래소 Binance의 공용 응용 프로그래밍 인터페이스 (API)에 대 한 변경은 회사가 9 개의 거래 쌍에 대 한 마진 거래를 가능 하 게 했다는 것을 밝혀 냅니다.

자체적으로 설명 하는 변수 이름은 Binance가 마진 거래 기능을 구현 하는 가운데에 있음을 암시 합니다.

Reddit 사용자는 5 월 1 일에 모든 쌍에 대해 이전에 비활성화 된 isMarginTradingAllowed 플래그가 이제 9 개의 거래 쌍에 대해 사용 하도록 설정 되었다는 것을 지적 했습니다.

Binance는 먼저 마진 거래가 현물 거래 후와 거래소에 선물 되기 전에 구현 될 것 이라는 백서의 기능 롤아웃 섹션에서 약속 했습니다.

하나의 Reddit 사용자가 게시물에 대 한 의견을 지적 하는 것 처럼 지금 새로운 것은 API의 실제 변화입니다.

Cointelegraph 3 월에 보고, 일본 금융 규제는 보도 암호 화폐 마진 거래에 대 한 새로운 규정을 도입 했다.

또한 2 월, 몰타 기반 주요 암호 화폐 거래소 OKEx는 4 개의 새로운 마진 거래 쌍을 최대 100 배 레 버리지를 가진 플랫폼에 추가 했습니다.
