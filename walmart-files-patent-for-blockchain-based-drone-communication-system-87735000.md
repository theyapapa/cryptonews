# 월마트, 블록체인 기반 드론 통신 시스템 특허 출원 ...

###### 2019-08-14 20:08

미국의 소매 업체 인 월마트는 블록 체인 기술을 기반으로하는 무인 항공기 통신 시스템에 대한 특허 출원을 제출했습니다.

월마트는 8월 1일에 미국 특허상표청(USPTO)에 특허를 출원했다.

이 서류는 드론의 작동 매개 변수를 암호화하고 저장하는 드론의 시스템을 설명합니다.

본질적으로, 무인 드론 사이의 차량 간 통신 시스템은 메시지를 전달하고 통신 네트워크를 메쉬하며 기계가 서로의 위치와 베어링을 인식할 수 있도록 합니다.

## 월마트의 블록체인 특허 진출

월마트는 적어도 2017 년부터 드론을위한 블록 체인 솔루션 특허에 종사해 왔습니다.

이전에 Cointelegraph에 의해보고 된 바와 같이, 월마트는 5 월에 USPTO에 특허 출원을 제기 2017 "안전한 위치를 무인 항공 배달" 시스템에 대한.

## IBM은 비슷한 목표를 가지고

이전에 Cointelegraph에 의해보고 된 바와 같이, IBM은 또한 블록 체인 기술을 통해 정보를 저장하고 공유하여 무인 항공기 통신을 보호 할 수있는 시스템에 대한 2017 년 USPTO와 특허 출원을 제출했다.
