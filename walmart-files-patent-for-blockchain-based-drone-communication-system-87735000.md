# Walmart Файлы патент для Blockchain основе дрон системы связи ...

###### 2019-08-14 20:08

Американский розничный гигант Walmart подал патентную заявку на систему связи дронов, основанную на технологии блокчейн.

Компания Walmart подала патент в Ведомство США по патентам и товарным знакам (USPTO) 1 августа.

Подача описывает систему на беспилотнике, который шифрует и хранит эксплуатационные параметры этого беспилотника.

По сути, система связи между беспилотными летательными аппаратами будет передавать сообщения, сетей связи сетки и позволит машинам быть в курсе местонахождения и подшипника друг друга.

-Предприятие Walmart по блокчейну

(Альт-Н)

Walmart занимается патентованием блокчейн-решений для дронов как минимум с 2017 года.

Как ранее сообщал Cointelegraph, walmart подала патентную заявку в ВПТО США в мае 2017 года на систему «Беспилотная воздушная доставка в безопасное местоположение».

IBM имеет аналогичные цели

(Альт-Н)

Как ранее сообщал Cointelegraph, IBM также подала патентную заявку в USPTO в 2017 году на систему, которая будет обеспечивать беспилотную связь путем хранения и обмена информацией с помощью технологий блокчейн.
