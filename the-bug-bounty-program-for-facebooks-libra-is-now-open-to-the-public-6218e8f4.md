# Ошибка Баунти Программа для Весов Facebook в настоящее время открыт для общественности ...

###### 2019-08-27 10:08

Ассоциация Весов открыла свою программу баунти для общественности для решения проблем безопасности в преддверии ожидаемого запуска Весов в 2020 году.

Пост подчеркнул, что расширение программы будет способствовать значения, критически важные для Ассоциации Весов, как "открытость, прозрачность и глобальный доступ", и обеспечить минимальные вопросы приходят даты запуска.

Ассоциация написала, что она ожидает открытия программы баунти до общественности будет поощрять "многие другие люди с различными навыками и опытом", чтобы проверить безопасность блокчейн АЙс.

Стежок во времени

(Альт-Н)

Другой 27 августа блоге на веб-сайте Весы Майкл Энгл, глава экосистемы разработчиков в Ассоциации Весов, сказал, что участники могут заработать до $ 10000 для выявления критических вопросов безопасности.

Регулятивные беды

(Альт-Н)

Примечательно, что Либра официальное объявление ошибка щедрость программы не упоминается о Facebook, организация, которую многие видят, как возглавляющий Ассоциацию Весов.

Частично в ответ на такие опасения, Дэвид Маркус попытался заверить общественность, что "Facebook будет только один из более чем ста членов Ассоциации Весов по запуску".

Регуляторные проблемы и проблемы безопасности были на переднем крае Весов и Facebook усилия по запуску своей новой цифровой валюты.
