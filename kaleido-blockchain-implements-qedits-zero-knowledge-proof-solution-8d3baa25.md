# Kaleido 블록 체인은 QEDIT의 제로 지식 증명 솔루션을 구현합니다. ...

###### 2019-09-15 04:09

엔터프라이즈 블록 체인 스타트업 인 Kaleido는 ConsenSys의 벤처 스튜디오의 일부로 암호화 개인 정보 보호 회사 인 QEDIT의 제로 지식 증명 솔루션을 구현한 최초의 기업이되었습니다.

## 선택적 기능

QEDIT는 Sept. 15에 Cointelegraph와 공유 한 보도 자료에서 Kaleido와의 파트너십이 플랫폼을 사용하는 기업을위한 상용 옵션으로 개인 자산 이전 기능을 가능하게 할 것이라고 주장합니다.

QEDIT 조나단 루치의 CEO 겸 공동 설립자는 다음과 같은 추가 를 추가했습니다.

"Kaleido는 기업이 사용자 정의 코드 개발에 상당한 자원을 투자 할 필요성을 제거하여 엔터프라이즈 블록 체인 기술의 배포를 가속화하는 데 선도적 인 역할을했습니다.

## 주목할 만한 플랫폼

Rouach는 또한 Komgo 블록 체인을 칼레이도의 잠재력의 예로 지목하여 "선도적 인 에너지 회사와 금융 기관에서 사용된다"고 주장했습니다.

발표에 따라, Kaleido는 "기업과 컨소시엄 준비 된 유일한 풀 스택 블록 체인 SaaS입니다."

코인텔레그래프가 지난해 5월 보도한 바와 같이, 선도적인 클라우드 플랫폼인 아마존 웹 서비스는 칼레이도와 제휴하여 고객에게 간소화된 블록체인 클라우드 플랫폼을 제공했습니다.

2018 년 11 월, 두 사람은 또한 기업이 블록 체인 솔루션을 구현하는 데 도움이되는 풀 스택 플랫폼을 출시했습니다.
