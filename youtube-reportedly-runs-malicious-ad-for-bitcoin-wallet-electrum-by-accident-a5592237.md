# 유튜브는 사고에 의해 비트 코인 지갑에 대 한 악의적 인 광고를 실행 보도 ...

###### 2019-03-26 16:03

비디오 공유 플랫폼 YouTube 소문는 3 월 26 일에 발표 된 Reddit post에 따라 실수로 비트 코인 (BTC) 지갑 전기를 위한 악성 광고를 실행 했습니다.

광고에 관심이 있는 시청자는 typos콰트로 팅 또는 URL 하이재킹 이라는 일반적인 사기 방법을 사용 하 여 악의적인 링크로 리디렉션됩니다.

Reddit 게시물에서, mrsxe, 라는 사용자는 전기의 악성 코드 버전의 승진에 대해 대 중에 게 경고 하 고 사기 광고가 작동 하는 방법을 설명 했다: "악의적 인 광고는 실제 전기 광고 처럼 보이게 변 장 된다 \ [... \] 그것은 심지어 당신을 알려줍니다 비디오에서 올바른 링크 (electrum.org)로 이동 하지만 광고를 클릭 하면 즉시 악성 EXE 파일을 다운로드 하기 시작 합니다.

2 월, 암호 화폐 지갑의 사용자는 전자 지갑 그들은 피싱 공격에 직면 했다 보고.

Reddit에 한 사용자는 전기 고객 으로부터 중요 한 데이터를 훔 치기 위해 시도 하는 피싱 사기가 보안 업데이트로 포즈를 취한 것을 발견 했습니다.

3 월 초에, 구글 크롬 브라우저 확장은 암호 화폐 거래소 Huobi에서 가짜 공중 투 하에 참여로 사용자를 속여 NoCoin을 불리는, 230 피해자를 주장.

해커는 고의로 암호 화폐 악성 코드 또는 소위 크립 토 도둑 으로부터 사용자를 보호 하는 도구 처럼 보이게 악성 확장을 위장 했다.
